# DevelopmentOS plugin

This Codex plugin exposes DevelopmentOS through the repository's MCP server. It deliberately contains no credentials. Set `DEVELOPMENTOS_TOKEN` and `DEVELOPMENTOS_URL` in the environment that launches Codex before installing or using it.

See [`scripts/developmentos-mcp/README.md`](../../scripts/developmentos-mcp/README.md) for setup, security, clients, tools, and tests.
